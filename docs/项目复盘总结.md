# 多算法波束形成仿真项目 — 复盘总结

> **项目**: beamforming-sim v0.1.0  
> **日期**: 2026-05-07  
> **定位**: 实验性声学波束形成仿真项目，不面向发布，不保留旧 API 兼容层  
> **环境**: Python 3.11+, NumPy, Matplotlib, SciPy  
> **测试**: 39 项，全部通过

## 一、项目做了什么

本项目模拟自由场近场点声源，用麦克风阵列信号生成声源定位图。默认场景是 64 阵元八臂螺旋阵列、9 个 25 kHz 单声源 case：3 个距离平面 `1.2/1.6/2.0 m`，每个平面 3 个横向位置 `0/+0.5/-0.5 m`。

当前算法：

| 算法 | 输入 | 作用 |
|---|---|---|
| CBF | CSM | 常规波束形成基线 |
| FB | CSM | 函数波束形成，`nu` 越大主瓣越窄 |
| FFT-FISTA | CBF dirty map | 平移不变 PSF 近似反卷积 |
| DAMAS-FISTA | CBF dirty map | 精确 PSF 算子 + FISTA-NNLS |

默认入口 `main.py` 跑 CBF、FB、FFT-FISTA。DAMAS-FISTA 成本更高，通过 `main.py --include-damas-fista` 或 `scripts/run_damas_fista_single_source.py` 显式运行。

## 二、核心链路

```text
点声源
  -> 自由场传播：时延 + 1/r 衰减
  -> 64 路麦克风信号
  -> 目标频率 CSM
  -> CBF / FB / FFT-FISTA / DAMAS-FISTA
  -> dB 热力图和指标
```

关键实现：

- `array_geometry.py` 生成螺旋阵列。
- `scene.py` 生成声源和扫描平面。
- `signals.py` 生成麦克风时域信号。
- `spectral/csm.py` 计算目标频率交叉谱矩阵。
- `algorithms/` 存放四种算法。
- `experiments/` 编排默认实验和 DAMAS-FISTA 单声源实验。

## 三、当前设计取舍

项目已按实验代码收敛：

1. **无发布目标**：不保留旧版函数式入口，不做兼容门面。
2. **弱防御校验**：默认输入来自本项目配置、脚本和测试；删除大部分 `ValueError` 参数保护。
3. **结果显式归一化**：算法返回 `BeamformingResult.raw_power`，绘图或比较时再调用 `normalized_power()` / `power_db()`。
4. **CSM 复用**：同一声源 case 的多算法共用 CSM，避免重复频谱计算。
5. **DAMAS-FISTA 显式运行**：默认 `0.01 m` 网格有 14641 个扫描点，不适合默认强制跑精确 PSF 反卷积。

## 四、模块职责

| 模块 | 职责 |
|---|---|
| `main.py` | 命令行入口 |
| `algorithms/base.py` | `Beamformer` 协议和 CSM 算法共用 `run()` |
| `algorithms/cbf.py` | CBF、导向矩阵、二次型功率 |
| `algorithms/fb.py` | FB 和 CSM 分数次幂 |
| `algorithms/fft_fista.py` | FFT-FISTA 反卷积 |
| `algorithms/damas_fista.py` | DAMAS-FISTA 精确 PSF 反卷积 |
| `domain/result.py` | `BeamformingResult` 结果容器 |
| `experiments/runner.py` | 默认 CBF/FB/FFT-FISTA 实验 |
| `experiments/damas_fista.py` | DAMAS-FISTA 单声源实验和指标 CSV |
| `experiments/writer.py` | 统一输出图像 |
| `visualization.py` | 热力图、阵列图、信号图 |

## 六、后续优化

优先级按当前目标排序：

1. 删除过期架构文档中已完成或不再适用的建议。
2. 给 `main.py --include-damas-fista --damas-step 0.1` 增加一条轻量集成测试。
3. 增加多声源 case，只在明确需要对比多声源分辨能力时做。
4. 若 DAMAS-FISTA 要跑更密网格，再考虑算子缓存或更快的近似策略。
